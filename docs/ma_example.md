# M&A 估值建模完整示例

> 以某矿业企业收购某矿业标的为例，演示 M&A 并购置入估值建模流程。

## 项目背景

- **收购方**：X公司（某矿业上市公司）
- **标的方**：Y公司（某钒钛铁矿标的）
- **交易类型**：发行股份购买资产
- **评估基准日**：2025年3月31日

## Step 1: 报告完备性检查

| 报告 | 状态 | 说明 |
|:---|:---:|:---|
| X公司近3年年报 | ✅ | 年报齐全 |
| 重组报告书 | ✅ | 含标的财务+协同+可比公司+估值参数 |
| Y公司审计报告 | ✅ | 2024年度+2025年1-3月 |
| 评估报告 | ✅ | 产能/价格/成本参数 |

## Step 5: Y公司 FCFF

### 5.1 产能与爬坡

| 参数 | 数值 |
|:---|:---|
| 产能 | 露采500万t/年, 地采200万t/年 |

### 5.2 产品与价格

| 产品 | 品位 | 年产量(万t) | 均价(元/t) |
|:---|:---:|:---:|:---:|
| 铁精矿 | 61% | 92.28 | 640.73 |
| 钛精矿 | 47% | 41.92 | 2,113.24 |
| 次钛矿 | 42% | 7.10 | 1,423.61 |

### 5.3 协同效应

```
含协同 EV = 688,762 万（估值报告）
收购对价 = 650,768.80 万
```

## Step 6: 备考合并 DCF

```
备考合并FCFF = X公司独立FCFF + Y公司FCFF + 协同增量
合并WACC：迭代求解（D/E变化 + 整合溢价0.3%）

合并每股价值 ~29.4元
```

## Step 7: 估值对比

| 情景 | 每股价值 |
|:---|:---:|
| 无并购 | 29.3元 |
| 备考合并 | 29.4元 |
| **差异** | **+0.1元** |

→ 并购微弱提升每股价值，协同效应基本被收购溢价消耗。

## M&A 重大资产出售示例

> 以A公司（某*ST企业）出售地产子公司为例。

### 出售类型判定

```
标的资产：17家并表房地产子公司100%股权
结论：✅ 并表子公司出售，DCF 建模适用
```

### 备考扣非分离

| 项目 | 含非口径 | 扣非口径 | 差异 |
|:---|:---:|:---:|:---:|
| 归母净利润 | +2.25亿 | -0.62亿 | 2.87亿（处置收益） |

> ⚠ 必须用扣非口径作为 DCF 起点

### 出售前 DCF 判定

```
归母权益: -17.5亿 → DCF 不适用
出售后备考归母权益: +4.9亿 → DCF 适用
```

### 备考对比

| 指标 | 出售前 | 出售后(备考) |
|:---|:---:|:---:|
| 每股净资产 | -X元 | +Y元 |
| 归母权益 | -17.5亿 | +4.9亿 |
| DCF适用性 | 不适用 | 适用 |
