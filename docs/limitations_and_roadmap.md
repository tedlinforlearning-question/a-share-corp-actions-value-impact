# 局限性与后续计划

> 写于 v1.2.0。任何工具都有边界，诚实面对边界比假装全能更有价值。

---

## 当前已知局限

### 1. 数据源依赖性

| 局限 | 说明 | 影响 |
|:---|:---|:---|
| Wind API 付费门槛 | Wind 终端需要机构账号，个人用户无法直接使用 | 限制了非机构用户的使用体验 |
| AKShare 数据覆盖不全 | β 值、部分可比公司财务指标不如 Wind 完整 | 免费方案下 WACC 计算精度可能降低 |
| 数据时效性 | 依赖用户上传的 PDF 报告，不自动拉取最新季报 | 估值基准日与当前时间可能差距较大 |

### 2. 报告解析局限

| 局限 | 说明 |
|:---|:---|
| **扫描版 PDF** | 如果审计报告是扫描图像而非文本，PyMuPDF 无法提取，需退回招股书或其他来源 |
| **格式差异** | 不同券商/公司的 PDF 排版差异大，关键词搜索可能漏掉部分数据 |
| **附注深度** | 仅提取主表和常见附注，不涉及复杂的会计政策调整（如同控合并、资产减值测试等） |

### 3. DCF 方法论局限

| 局限 | 说明 |
|:---|:---|
| **终值假设敏感** | 永续增长率 g 和 WACC 的微小变化会大幅影响终值，这是 DCF 方法本身的固有问题 |
| **线性外推假设** | 收入增速、EBIT 率等参数假设路径较线性，无法处理结构性变化（如行业拐点、政策冲击） |
| **单一估值方法** | 仅提供 DCF 估值，未集成可比公司法（PE/PB/EV/EBITDA）作为交叉验证 |
| **收购+配套场景** | 新完成（v1.2.0），关联交易抵消和协同效应量化的标准化程度仍有提升空间 |

### 4. 分享使用的平台依赖风险 ⚠

> **以下是分享给别人使用前必须注意的云端依赖问题：**

| 依赖项 | 风险 | 解决方案 |
|:---|:---|:---|
| **wind-mcp-skill** | 接收方需要单独安装此 Skill + 拥有 Wind API Key | SKILL.md 已将 Wind 标记为可选，提供手动输入/akshare 替代路径 |
| **Wind API Key** | 需要 Wind 机构账号在 aifinmarket.wind.com.cn 创建 | 技能明确说明"无云端数据源也能完成全部分析" |
| **GitHub raw 链接** | Excel 模板托管在 GitHub，需能访问 raw.githubusercontent.com | 同时提供本地 `python scripts/build_dcf_model.py --all` 生成方式 |
| **WorkBuddy 平台** | Skill 基于 WorkBuddy/CodeBuddy 的 skill 加载机制 | README 提供了独立 Python 脚本使用方法 |
| **AKShare** | 免费无需注册，但首次调用需联网下载缓存 | 覆盖面和更新频率不如 Wind |
| **Tushare** | 免费但需注册获取 Token | 作为 AKShare 之外的备选 |

> **核心分析（DCF建模、摊薄分析、备考合并）不需要任何云端数据源。** 仅在 WACC 校准和可比公司环节才需要外部数据，且可手动输入替代。

### 5. 行业适应性

| 行业类型 | 适用性 | 备注 |
|:---|:---|:---|
| 制造业（IC封测、矿业等） | ✅ 适用 | 有在建工程/产能爬坡数据，DCF 框架完善 |
| 金融业（银行、保险） | ⚠ 需要调整 | 银行 DCF 使用股利折现模型（DDM），与当前 FCFF 框架不同 |
| 互联网/软件 | ⚠ 需要调整 | 轻资产模式，NWC 和 CAPEX 假设需大幅修改 |
| 生物医药（无收入） | ❌ 不适用 | 无收入阶段更适合 rNPV 或实物期权法 |
| 房地产 | ⚠ 需要调整 | 更适合 NAV（净资产价值法）或剩余收益模型 |

### 6. 使用流程局限

| 局限 | 说明 |
|:---|:---|
| **需要人工判断** | 收入增速、EBIT 率等核心假设仍依赖使用者经验，Skill 提供框架但不替代专业判断 |
| **分步交互耗时** | 采用逐步确认模式确保严谨性，但对于经验丰富的分析师可能感觉"太慢" |
| **单一语言** | 仅支持中文 PDF 报告（关键词匹配基于中文），英文 IPO 招股书（如港股、美股）需改造 |

## 后续计划

### 近期（v1.3 - v1.5）

- [ ] **可比公司估值法**：集成 PE / PB / EV/EBITDA 倍数法，与 DCF 交叉验证
- [ ] **银行 DDM 支持**：新增股利折现模型，覆盖金融行业 IPO 和并购
- [ ] **自动敏感性图表**：在 Excel 输出中直接生成 WACC×g 热力图
- [ ] **英文报告支持**：关键词库双语化，支持港股/美股 IPO 招股书

### 中期（v1.5 - v2.0）

- [ ] **行业参数模板库**：预设 IC 封测、矿业、医药、软件等行业的 DCF 默认参数
- [ ] **情景分析**：支持乐观/基准/悲观三情景并行建模
- [ ] **可转债场景**：扩展至可转债的稀释效应和转股影响分析
- [ ] **实时期限结构**：自动获取最新国债收益率曲线作为 Rf 输入

### 远期（v2.0+）

- [ ] **AI 辅助假设设定**：基于行业研报和历史 DCF 案例，自动推荐参数区间
- [ ] **多项目组合分析**：同时评估多个募投/并购标的，自动生成最优组合
- [ ] **合规检查模块**：根据 IPO/重组审核口径，自动校验假设合理性和披露完备性
